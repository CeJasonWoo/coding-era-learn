## 大O记法
>（n表示数据规模， “O”表示量级 ）

    O(1) 常数阶 我的理解(与n无关)
    O(n) 线性阶 我的理解(单个for循环)
    O(n^2) 平方阶 我的理解(2重for循环嵌套)
    O(n^3) 立方阶 我的理解(3重for循环嵌套)
    O(log2n)  对数阶 我的理解(递归)
    O(n*log2n)  线性对数阶
    k次方阶O(n^k)
    指数阶O(2^n)

> 其中 O(2^n) 不实用， O(log2n) 在n比较大时效率较高
